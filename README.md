# RainloxsTrading

A plugin maded for minecraft server to trade items more realisticly.
### WARNING
Needs EssenstialsX to pay money and other fancy things

## Use
Command /trade {player name} gives item stack from your MAIN hand to a specified player, he must be near you.

## Usage
This plugin is for players, and it's about human honesty and it does not guarantee the integrity of every transaction.
Enjoy scaming at your server!
